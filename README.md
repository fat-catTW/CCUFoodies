# CCUFoodies
起源:
來中正大學就讀後，發現嘉義其實有很多精緻、好吃、適合約會的隱藏店家。 但外地來的學生不一定知道。 相要透過這個專案，讓在中正就讀的學生們在用膳上更又選擇。

打算用 Line 官方帳號作為互動接口。
功能要有:
透過聊天，幫助使用者尋找特定類型的餐廳。
隨機抽取店家的功能。
內部評分系統，使用者能幫店家評分，在未來給他人參考。


需要的資料:
Google Map 上店家的地址
餐廳分類: 中式、日式、韓式、泰式、火鍋、燒烤、餐酒館、冰品、飲料、咖啡廳、咖哩、麵包、早午餐、酒吧、鐵板燒、甜品、炸物、牛排、越式


前端:
Line



後端:
伺服器: Flask
資料庫:
